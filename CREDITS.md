# Credits & Attributions

Talon Flow is a specialized "Super Agent Harness" for autonomous career automation and recruitment intelligence.

## Base Architecture
This project is built upon the robust multi-agent foundations of [bytedance/talon-flow](https://github.com/bytedance/talon-flow). We extend our thanks to the original authors for their world-class agents and sandbox implementations.

## Talon Flow 3.5 Enhancements
The following enterprise-grade features have been built by **Thousifuddin Shaik** to extend the original architecture:

- **Vector RAG Intelligence**: Integrated a local Vectra-based job intelligence engine for semantic search across 600+ scraped leads.
- **Swarm Orchestration**: Refactored the core lead engine into a 3-agent autonomous swarm (Filter, Scout, Architect).
- **Secure Cloudflare Tunnelling**: Implemented persistent zero-trust tunnels to bridge local recruitment agents to the public Talon Flow UI.
- **Mercury Filter Logic**: Advanced filtering for F-1 OPT eligibility and visa-safe recruitment targeting.
- **Recruiter Sync Control**: Automated resume synthesis and recruiter outreach sequencing.

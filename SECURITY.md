# Security Policy

## Supported Versions

As talon-flow doesn't provide an official release yet, please use the latest version for the security updates.
Currently, we have two branches to maintain:
* main branch for talon-flow 2.x
* main-1.x branch for talon-flow 1.x 

## Reporting a Vulnerability

Please go to https://github.com/bytedance/talon-flow/security to report the vulnerability you find.
